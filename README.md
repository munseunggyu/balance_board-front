## 팀스페이스 진행방식

스프린트(2주) > 프로젝트(1~2주) > 테스크

- 스프린트 : 2주 활동의 대분류
- 프로젝트 : 스프린트에 속하는 활동별 주제 중분류
- 테스크 : 각 프로젝트를 수행하기 위한 담당자별 할일 리스트
- 회고 : 2주 스프린트 이후에 목표 및 협업에 대해 회고

\*2주 단위로 KPI 목표 설정 > 회고
